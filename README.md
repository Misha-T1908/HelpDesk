# HelpDesk Pro

HelpDesk Pro — україномовний веб-додаток на Flask + SQLite для обліку, розподілу та контролю заявок користувачів до IT-відділу.

Проєкт створений як практичний результат навчальної практики з комп'ютерних наук. Його мета — показати реалістичну невелику внутрішню helpdesk-систему, а не просту форму збору даних.

## Можливості

- авторизація користувачів;
- ролі: користувач та адміністратор;
- створення заявок через спрощену зручну форму;
- додаткові дані заявника, підрозділу та пристрою;
- категорії, пріоритети та статуси заявок;
- призначення виконавця на сторінці заявки;
- коментарі до заявки;
- панель заявок із пошуком і фільтрами;
- аналітичні звіти;
- експорт CSV;
- база даних SQLite;
- автоматичне створення/оновлення структури бази даних при запуску.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Після запуску відкрити у браузері:

```text
http://127.0.0.1:5000
```

## Запуск через Gunicorn

Для хостингу на сервісах на кшталт Render можна використовувати команду:

```bash
gunicorn app:app
```

Рекомендована змінна середовища для продакшн-запуску:

```text
SECRET_KEY=your-long-random-secret
```

## База даних

Структура SQLite створюється та оновлюється автоматично під час запуску застосунку. Для ручної ініціалізації можна виконати:

```bash
flask --app app init-db
```

Якщо потрібно безпечно пересоздати базу з тестовими даними для демонстрації або захисту практики:

```bash
flask --app app reset-db
```

Для автоматичного підтвердження пересоздання:

```bash
flask --app app reset-db --yes
```
